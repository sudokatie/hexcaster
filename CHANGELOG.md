# Changelog

## [0.1.0] - 2026-01-30

### Added
- Hex grid system with 6-direction movement
- Procedural dungeon generation with rooms and corridors
- ECS architecture using hecs
- Combat system with melee attacks
- Enemy AI that chases and attacks player
- Rune collection and spell crafting
- Multiple spell shapes (Point, Line, Cone, Ring, Burst)
- TUI rendering with ratatui
- Health, AP, and inventory display
- Win/lose conditions
- 63 tests
